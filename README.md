## Course Assessment

1. 平时出勤：占考核10%。
2. 论文分享：占考核20%。
  - 从11月5日起，论文分享，每人10分钟，请大家准备好PPT，主要归纳阐述论文的研究背景，创新点，相关领域的研究方法，论文的方法，实验效果，案例分析，启发，需要高度提炼。论文的选择可以从给出的列表中选择，也可以选择和知识图谱相关自己感兴趣的文章。
3. 项目占考核70%
  - 完成一个知识图谱相关的任务算法、工具或者原型系统，提交和检查实践代码，占考核40%；1月14日和1月15日检查代码。
  - 围绕项目写一篇报告，从研究或开发背景，相关研究，采用的方法，创新点和特色，实验评测，系统功能和实现等方面介绍，占考核30%
